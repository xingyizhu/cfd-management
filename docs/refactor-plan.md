# CFD 管理系统重构计划

## 目标

- 后端迁移到 `Spring Boot + Maven`
- 前端迁移到 `Vue 3 + Vite`
- 本期仅迁移核心报表链路：成员、同步、日/周/季聚合、范围汇总、工时不足提醒
- 继续使用 Supabase 托管的数据表
- 系统按内部工具设计，暂不做登录

## 范围

### 后端

- 提供成员列表、范围汇总、日/周/季明细、区间同步、提醒预览与发送接口
- 封装 Atlassian/Jira 拉取、节假日处理、聚合逻辑、缓存写入与 SMTP 提醒
- 增加定时任务开关与 Cron 配置

### 前端

- 提供日期范围选择、手动刷新、核心指标、成员汇总、日/周/季表格、明细表、提醒发送入口
- 使用前后端分离方式访问后端 API

## 实施顺序

1. 完成 Spring Boot 骨架和配置清理
2. 迁移 Python 核心聚合与同步链路到后端服务
3. 提供核心报表与提醒 API
4. 搭建 Vue 3 前端并接入后端
5. 通过 Maven 测试与前端构建完成静态验证

## 当前约束

- 仓库里已有未提交的 Spring Boot 初稿，实施过程中保留并扩展，不做回退
- Python CLI 与 Streamlit 代码暂保留，作为迁移期参考实现
